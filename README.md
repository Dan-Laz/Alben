# Alben

## Descrizione
Un'applicazione che consente di visualizzare mappe 3D sia in modo online che offline per la navigazione di sentieri in montagna. L'applicazione utilizza dati OSM (OpenStreetMap) tramite l'Overpass API per ottenere informazioni su rifugi e sentieri, e dati DEM (Digital Elevation Model) da SRTM (Shuttle Radar Topography Mission) per la rappresentazione del terreno. L'applicazione può collegarsi a un server per scaricare temporaneamente la mappa per l'uso online o permanentemente per l'uso offline. È presente un piano gratuito (funzionalità base) e un piano a pagamento (funzionalità completa).

## Target
Escursionisti / Alpinisti

## Problema Risolto
L'app elimina il rischio di perdersi in montagna, causato dalla poca segnaletica.

## Tecnologie e Dati
* OpenStreetMap (OSM)
* Overpass API (OSM)
* SRTM DEMs
* Server centrale che gestisce le "tiles" della mappa

## Concorrenti
* Strava (fatmap)
* Gaia GPS
* AllTrails

## Tagline
Explore the world! without getting lost.
